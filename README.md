# 💰 CoinPulse

**CoinPulse** is a modern cryptocurrency tracking and analytics platform designed to deliver real-time market insights in a clean, fast, and intuitive interface. It helps users monitor price movements, track market trends, and stay updated with the crypto ecosystem without noise or clutter.

---

🔗 Live Demo: https://coinpulse-phi.vercel.app/

## 🧱 Tech Stack

- Frontend: React
- Styling: CSS  
- Charts: Recharts / Chart.js  
- API: CoinGecko

---

## 🛠️ Getting Started

```bash
git clone https://github.com/your-username/coinpulse.git
cd coinpulse
npm install
npm run dev
```
